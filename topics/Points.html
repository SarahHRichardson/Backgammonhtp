Points

Game Point Values

Players who plan to play more than one game of backgammon may assign varying point values to the games. These point values are determined by the types of wins and the doubling die.

Types of Wins

There are three types of wins in Backgammon. The first win is the standard victory in Backgammon, where both players have taken at least one chip off the backgammon board. This win is worth 1 point.



The second type of win is called a gammon. This type of win occurs when the losing player has not removed any chips from their board. This win is worth 2 points.



The final type of win is called a backgammon. This type of win occurs when, not only has the losing player not removed any chips, they still have chips in the winner's home quadrant. This win is worth 3 points.



Doubling Cube

The doubling cube/die is the die labeled 2, 4, 8, 16, 32, 64.

At the start of their turns, each player can try to double the value of the game by asking their opponent. If the player's opponent accepts, turn the doubling die to the point value of the game thus far (2 if the game is doubled once, four if the game is doubled twice, and so on). The most that the value of a game can be doubled is six times for a value of 64 points. If the player's opponent refuses, they forfeit the game, giving the current game value to the game's winner. For example, if a player refuses to double the value of a game currently worth 2 points to 4 points, their opponent wins 2 points from the game and the game ends.

Total Points

At the end of a fully-played game, multiply the game's doubling value (1-64) by the type of win value. For example, if the game was doubled once to a point value of 2 and the player wins with a gammon (2 points), the player receives 2 x 2 = 4 points.
